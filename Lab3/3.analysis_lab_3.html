Аналіз задачі

Завдання 1:

1. Для розрахунку першої суми з параметром від 1 до 5 використовували цикл for .

2. Для розрахунку внутрішньої суми з параметром від 0 до нескінченності використовували цикл while або do... while.

3. Для визначення нескінченності задали точність розрахунку значення поточного элементу ряду. Підсумовування припиняється, якщо досягнута задана користувачем точність розрахунку.

4. Кількість доданків внутрішньої суми для кожного параметра зовнішньої суми має бути різною.

5. Була створена функція для обчислення факторіалу.

6. Для запобігання переповнення (overflow ) комірки RAM здійснити вихід із внутрішнього циклу при досягненні значення k>30 .
